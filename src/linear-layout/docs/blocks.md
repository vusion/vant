### 单行排布

``` html
<van-linear-layout direction="horizontal">
</van-linear-layout>
```

### 多行排布

``` html
<van-linear-layout direction="vertical">
    <van-linear-layout>
    </van-linear-layout>
    <van-linear-layout>
    </van-linear-layout>
</van-linear-layout>
```

### 两端排布

``` html
<van-linear-layout justify="space-between">
    <van-linear-layout>
    </van-linear-layout>
    <van-linear-layout>
    </van-linear-layout>
</van-linear-layout>
```
